# نظام حساب تكلفة الموظفين التفاعلي (تكلفتي)

تطبيق ويب متكامل ومتعدد الشركات (Multi-Company) لحساب **التكلفة الحقيقية والفعلية للموظفين (Fully-Burdened Cost)** شهرياً وسنوياً، وتوزيع عبء السعودة وتكاليف الإقامة والعمل، بناءً على النموذج المالي لملف الإكسيل.

---

## ✨ المميزات الرئيسية
1. **تعدد الشركات (Multi-Tenancy):** إمكانية تسجيل كل شركة بحساب مستقل وآمن، ومعاملات مالية معزولة تماماً.
2. **محرك الحسابات المالية (Costing Engine):** حساب مستحقات التأمينات (GOSI)، ومخصص الإجازات، ومكافأة نهاية الخدمة، وتذاكر الطيران، ورسوم الإقامة، ورخص العمل.
3. **توزيع السعودة الذكي (Saudization Allocation):** حساب إجمالي تكاليف الموظفين السعوديين وتوزيعها كنسبة عبء مالي شهري بالتساوي على جميع الموظفين المقيمين بشكل ديناميكي.
4. **استيراد ملفات الإكسيل (Excel Import):** إمكانية رفع شيت موظفين ورواتب مباشرة وسحب البيانات وتحديث قاعدة البيانات تلقائياً.
5. **لوحة تحكم تفاعلية (Interactive Dashboard):** مؤشرات مالية رئيسية (KPIs)، ومخططات رسوم بيانية توضح توزيع التكاليف والرواتب، ورسم بياني توضيحي لنسب التوطين.
6. **التقارير التجميعية للمشاريع (Pivot Reports):** صفحة تقارير تجميعية شاملة للمشاريع والفروع مطابقة للجدول المحوري (Pivot Table) في ملف الإكسيل مع إمكانية التصدير بصيغة CSV.

---

## 🛠️ متطلبات التشغيل والتشغيل المحلي

### 1. تشغيل الواجهة الخلفية (Backend API)
من موجه الأوامر أو الطرفية، انتقل إلى مجلد `backend`:
```bash
cd backend
npm install
npm start
```
* سيتم تشغيل الخادم على المنفذ `5000` (`http://localhost:5000`).
* سيتم إنشاء ملف قاعدة بيانات خفيف وتلقائي باسم `database.json`.

### 2. تشغيل الواجهة الأمامية (React Frontend)
افتح نافذة أوامر جديدة وانتقل إلى مجلد `frontend`:
```bash
cd frontend
npm install
npm run dev
```
* سيتم تشغيل الواجهة الأمامية محلياً وعرض رابط التشغيل (غالباً `http://localhost:5173`).

## 📋 نموذج لتجربة استيراد ملف الإكسيل
عند تشغيل التطبيق وإنشاء حساب شركة جديد، انتقل إلى صفحة **استيراد الإكسيل** لتعبئة البيانات تلقائياً.
* يدعم النظام رفع شيت إكسيل يحتوي على الأعمدة التالية (بالعربية أو الإنجليزية):
  * `الرقم الوظيفي` أو `الرقم`
  * `اسم الموظف` أو `أسم الموظف`
  * `الجنسية` (مقيم أو سعودي)
  * `الراتب الأساسي`
  * `بدل السكن`
  * `بدل الانتقال`
  * `بدل معيشة`
  * `بدلات أخرى`
  * `التأمين الطبي`
  * `الشهادة الصحية`
  * `تأشيرة الخروج والعودة`
  * `المشروع` أو `الموقع`
